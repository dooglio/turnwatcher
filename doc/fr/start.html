Votre premier combat!

Maintenant vous êtes prêt pour démarrer votre premier combat avec Turn Watcher!

  • Démarrez le logiciel et abandonner tous les personnages de la liste si nécessaire (Abandonner est dans le menu Fichier).
  • Sélectionnez Importer du menu Fichier pour charger votre liste de personnages participant à votre campagne.
  • Sélectionnez Importer à nouveau pour charger la liste des monstres pour cette rencontre. Turn Watcher vous demandera si vous désirez fusionner avec votre liste existante (les personnages). Cliquez sur Oui.

  • Démarrez votre combat en cliquant sur le bouton Démarrer de la barre d'outils (remarquez que quand vous cliquez sur le bouton Démarrer il se transforme en un bouton Terminer). Turn Watcher lance les dés pour l'initiative de façon automatique et ceci pour tous les personnages dans la liste. Ensuite il trie les personnages pour vous! Enfin, il montre le numéro du round en court dans la barre de status au bas de la fenêtre.
  • Pressez sur le bouton Suivant pour déplacer le curseur d'un personnage à l'autre quand le personnage actuel à fini son action. Le personnage dont c'est le tour est pointé par une flêche (en forme d'épée) sur la gauche du nom du personnage.

Idée: Utilisez la barre d'espace pour facilement avancer d'un personnage à l'autre.

  • Pour arrêter votre combat, cliquez sur le bouton Terminer.


Idée: À n'importe quel moment, si vous faites une erreur, vous pouvez utiliser la fonction Annuler dans le menu Édition.
