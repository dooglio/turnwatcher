Commençons...

C'est facile de voir que Turn Watcher est un outil très utile dans votre armurie de maître de jeu. Les bases sont:

  1. Créer une liste de personnages joueurs impliqués dans un combat (vous pouvez ré-utiliser cette liste encore et encore au fur et à mesure de vos rencontres).
  2. Créer des listes de monstres.
  3. Importer la liste des PJs et l'une des listes de monstres pour votre rencontre.
  4. Démarrer votre combat!

Ajouter des personnages joueurs

La toute première fois que vous démarrez Turn Watcher, vous verrez une liste vide.

Commencez par entrer les statistiques pour chacun de vos personnages joueurs (PJs). Nous vous recommandons de vous préparer (de créer vos listes) avant votre partie. Il ne vous est pas nécessaire d'entrer les statistiques à chaque partie puisque Turn Watcher les mémorise. Vous pouvez aussi mettre à jour cette information quand les personnages changent de niveau.

Pour commencer votre liste de personnages, cliquez sur le bouton Ajouter sur la barre de menu. Le dialogue Ajouter un personnage apparait. Entrez son nom, ses ajustements d'Initiative, de Détection, d'Écoute et de Sauvegarde. Ensuite, tapez le nombre maximum de points de vie. Enfin, choisissez Personnage dans le menu et cliquez sur OK.

Répétez ces étapes jusqu'à ce que vous ayez entrer tous les personnages dans Turn Watcher.

Idée: Tapez la touche TAB pour passer d'un champ à l'autre.

Doug entre les statistiques pour Shiena, une barbare humaine. Il clique sur le bouton Ajouter et tape Shiena dans le champ réservé pour le nom du personnage. Il avance à la case suivante avec la touche TAB et tape les ajustements pour son Initiative, sa Détection, son Écoute, sa sauvegarde et enfin ses points de vie. Il laisse les Dommages actuels à 0 et s'assure que le Type de personnages est bien Personnage. Pour finir, il clique sur OK.

Exporter une liste de personnages joueurs

Une fois que vous avez entré tous les PJs qui participent à votre campagne dans Turn Watcher, vous pouvez Exporter votre liste. Choisissez Exporter dans le menu Fichier. Tapez un nom de fichier et cliquez sur le bouton Sauver.

Doug maîtrise une campagne avec quatre joueurs. Il entre les statistiques de chacun des quatres personnages et Exporte dans un fichier nommé Campagne du Dragon (Dragon Campaign).

Créer les rencontres

Effacez tous les personnages et monstres de votre liste en sélectionnant Abandonner dans le menu Fichier. Assurez-vous de bien exporter vos personnages joueurs avant de tout effacer.

Ajouter des monstres pour vos rencontres est similaire à ajouter des PJs, mais il vous faut choisir Monstre au lieu de Personnage dans le menu du bas.

Pour chaque rencontre, ajouter le ou les differents monstres qui attaqueront les PJs. Exporter chaque groupe de monstres dans un fichier avec un nom approprié pour cette rencontre.

Idée: Pour rapidement dupliquer un monstre, cliquez sur lui dans la liste et sélectionnez Dupliquer dans le menu Édition. Turn Watcher vous demande combien de fois vous désirez dupliquer ce monstre. Et vous pouvez même faire ça durant un combat!

Doug a prévu une rencontre avec quatre ghouls...
  • Il clique le bouton Ajouter et type Ghoul 1 dans le champ Nom du personnage,
  • Il entre les ajustements d'une ghoul: Initiative, Détecter, Écouter et Sauvegarde; puis les points de vie maximum. Il s'assure que le Type de personnage est bien mis sur Monstre. Et pour finir il clique sur OK pour mettre la Ghoul dans la liste.
  • Il sélectionne Ghoul 1 dans la liste en cliquant dessus et choisi Dupliquer dans le menu Édition.
  • Dans la fenêtre, il entre le nombre 3 et clique sur OK.
Maintenant, il a quatre ghouls pour attaquer ses joueurs.

Exporter une liste de monstres pour une rencontre

Maintenant que vous avez une liste de monstres, vous pouvez la sauver dans en fichier en utilisant Exporter dans le menu Fichier. Dans le dialogue de sauvegarde tapez un nom de fichier et cliquez sur Sauver.

Doug choisit Exporter dans le menu Fichier et il exporte ça liste de 4 Ghouls dans un fichier nommé Rencontre avec les Ghouls (Ghoul Encounter). Ensuite il sélectionne Abandonner aussi dans le menu Fichier, et il recommence avec les monstres de la rencontre suivante.

Répètez ces étapes pour chaque rencontre que vous avez prévu dans votre prochaine partie.

Idée: Créez quelques listes de monstres pour des rencontres improvisées à utiliser pour aggrémenter vos parties.
